Question-1: What are the different ways to select an element in the DOM?

Answer : We can use methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector,
and querySelectorAll to select elements in the DOM.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1.innerHTML gets or sets the HTML content inside an element.
2.innerText returns only the visible text, ignoring hidden elements.
3.textContent returns all text, including hidden elements, without HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you add a single event listener to a parent element to handle events for its child elements, improving performance and reducing code.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and propagates up through its ancestors in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

1.Create: Use document.createElement('tagName').
2.Add: Use appendChild() or insertBefore() to add elements to the DOM.
3.Remove: Use removeChild() or remove() to delete elements.
